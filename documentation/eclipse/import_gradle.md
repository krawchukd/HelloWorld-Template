# Import Gradle

## Initial Steps

Obtain a copy of the repository, either as a zip or through a source control client.

# Project Import

## Open Eclipse

## In the file menu select ```import``` from the dropdown list.
![Project Import](../resources/file_menu_import.png)

## Select gradle project from the import list.
![Project Import List](../resources/select_import_type.png)

## Browse to project path
![Project Path](../resources/gradle_project_path.png)

## Verify Gradle Wrapper is selected.
![Gradle Wrapper](../resources/gradle_import_options.png)

## Project Layout View. Select Finish.
![Project Layout View](../resources/gradle_project_view.png)
